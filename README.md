# 🤖 Group Late Bot — Инструкция пользователя и описание системы

**Group Late Bot** — это умный, легкий и полностью автоматизированный Telegram-бот, интегрированный с системой учета рабочего времени **Workpace API**. Бот разработан для оперативного контроля дисциплины сотрудников и автоматического формирования детальной аналитики посещаемости за любой день.

---

## 🌟 Основные возможности

### 1. Оперативные оповещения в реальном времени (каждые 2 минуты)
Бот постоянно проверяет активность сотрудников по их расписаниям и присылает мгновенные карточки нарушений в рабочие чаты:

*   **🚨 Опоздание:** Если сотрудник отметился на вход позже времени начала своей смены (с учетом заданного порога, например, более 1 минуты).
*   **🏃 Ранний уход:** Если сотрудник отметился на выход раньше запланированного окончания смены.
*   **🚪 Отсутствует на месте:** Если смена сотрудника уже началась, но в течение **10 минут** отметка о приходе так и не появилась.
*   **⚠️ Забыл отметиться на выход:** Если смена сотрудника закончилась, но в течение **10 минут** после окончания смены отметка о выходе отсутствует.
*   **📸 Подозрительная отметка:** Если сотрудник приложил фотографию со статусом `0` (система подозревает фрод/обман или отметка ожидает ручной проверки). В этом случае в чат присылается карточка с именем устройства, временем и типом отметки.

### 2. Кнопка «Отбито» (Интерактивная обработка)
Каждая карточка нарушения снабжена кнопкой **«Отбито»**. 
*   Любой ответственный администратор может нажать её прямо в чате Telegram.
*   Бот мгновенно перепишет текст сообщения, добавив зеленую галочку, имя администратора, который нажал кнопку, и время отбивки.
*   Это позволяет координировать действия руководителей прямо в чате без ведения сторонних таблиц задач.

### 3. Интеллектуальный Excel-отчет (`/report`)
Бот умеет генерировать красивую Excel-таблицу за сегодня (или любую выбранную дату) и отправлять её в чат в виде документа. Отчет выполнен в премиальном корпоративном стиле и содержит следующие колонки:
1.  **Отдел** — название отдела из Workpace.
2.  **ФИО сотрудника** — имя сотрудника.
3.  **График** — название установленного расписания.
4.  **Все отметки за день** — полная хронология физических прикосновений с иконками: приход `(Вход)`, уход `(Выход)`. Если отметка подозрительная, рядом с ней выводится знак `⚠️`.
5.  **Время прихода (план)** — плановое начало смены.
6.  **Время прихода (факт)** — фактическое время первого входа.
7.  **Опоздание (мин)** — количество минут опоздания (при раннем приходе выводится прочерк `—`).
8.  **Время ухода (план)** — плановое окончание смены.
9.  **Время ухода (факт)** — фактическое время последнего выхода.
10. **Ранний уход (мин)** — количество минут раннего ухода.
11. **Отработано (факт) (HH:MM)** — реальное время, проведенное на работе (между самым ранним входом и самым поздним выходом за сутки).
12. **Отклонение (HH:MM)** — разница между фактическим временем работы и запланированной нормой смены:
    *   `+ЧЧ:ММ` — переработка (перевыполнение нормы).
    *   `-ЧЧ:ММ` — недоработка (уход раньше или опоздание).
    *   `+00:00` — ровно по норме.
13. **Статус** — итоговый статус посещаемости с цветовым выделением:
    *   🟩 **Вовремя** (светло-зеленый цвет) — пришел без опозданий.
    *   🟥 **Опоздал / Ранний уход / Отсутствует** (светло-красный цвет) — зафиксированы нарушения или неявка.
    *   🟨 **Вне графика / Вовремя (⚠️)** (светло-желтый цвет) — работа в выходной или наличие подозрительных отметок, требующих внимания.

## 🛠️ Умные внутренние алгоритмы (Self-Healing & Safety)

В боте реализованы уникальные защитные алгоритмы, исключающие сбои и ложные срабатывания:

1.  **Защита от ложного статуса «Отсутствует» (Алгоритм самолечения):**
    В Workpace, если сотрудник пришел слишком рано или его отметка признана подозрительной, API не связывает её со сменой, из-за чего стандартное поле фактического прихода присылается пустым. Бот автоматически распознает это, находит его физическую сырую отметку входа и берет её за основу. Это **полностью исключает ложные обвинения** в отсутствии сотрудника.
2.  **Фильтрация уволенных сотрудников:**
    Бот автоматически сканирует базу и полностью игнорирует архивных (уволенных) сотрудников, исключая их из Excel-отчетов и оперативных рассылок.
3.  **Блокировка дубликатов (Deduplication):**
    Поскольку выгрузка тяжелых отчетов требует времени (3-5 секунд), Telegram может повторно отсылать запросы. Бот снабжен кэшем `PROCESSED_UPDATES` и **никогда не отправит файл дважды** на один и тот же клик.
4.  **Игнорирование старой очереди сообщений:**
    При перезапуске сервера бот автоматически отбрасывает сообщения, отправленные пользователями более 60 секунд назад, отвечая только на «живые» команды в реальном времени.

## 💬 Команды бота в Telegram

*   `/start` — Приветствие, вывод вашего уникального Chat ID и списка доступных команд.
*   `/report` — Сгенерировать Excel-отчет со статистикой за **сегодняшний день** (по всем отделам).
*   `/report [Отдел]` — Сгенерировать отчет за **сегодня**, отфильтрованный по конкретному отделу (например, `/report КЦ`).
*   `/report ГГГГ-ММ-ДД` — Сгенерировать отчет за **любую дату** (например, `/report 2026-05-18`).
*   `/report ГГГГ-ММ-ДД [Отдел]` — Сгенерировать отчет за выбранную дату по конкретному отделу (например, `/report 2026-05-18 КЦ 3`).
*   `/report ГГГГ-ММ-ДД ГГГГ-ММ-ДД` — Сгенерировать **отчет за период** (диапазон дат) по всем отделам (например, `/report 2026-05-18 2026-05-20`).
*   `/report ГГГГ-ММ-ДД ГГГГ-ММ-ДД [Отдел]` — Сгенерировать **отчет за период по конкретному отделу** (например, `/report 2026-05-18 2026-05-20 КЦ 3`).
    *   *Примечание: первый лист отчета содержит сводную агрегированную аналитику по каждому сотруднику за период, а последующие листы — детальные ежедневные данные.*
*   `/addchat [id]` — *(Только для Главного Администратора)* Добавить новый чат (группу) в список автоматической рассылки опозданий. При добавлении бот отправит приветствие и список команд в этот чат, проверив доступность.
*   `/delchat [id]` — *(Только для Главного Администратора)* Удалить чат из списка рассылки.
*   `/chats` — *(Только для Главного Администратора)* Просмотреть список всех чатов, куда сейчас включена автоматическая рассылка.
